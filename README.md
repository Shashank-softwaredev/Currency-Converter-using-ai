# 📝 Todo App

A beautiful and functional Todo App built with vanilla HTML, CSS, and JavaScript.

## Features

✨ **Core Functionality:**
- ✅ **Create Tasks** - Add new tasks with a simple input field
- ✏️ **Edit Tasks** - Click the Edit button to modify existing tasks
- 🗑️ **Delete Tasks** - Remove tasks you no longer need
- ☑️ **Mark Complete** - Check off completed tasks
- 💾 **Local Storage** - Your tasks are saved automatically in your browser

🎯 **Additional Features:**
- 🔍 **Filter Tasks** - View All, Completed, or Pending tasks
- 📊 **Statistics** - See total, completed, and pending task counts
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations
- ⌨️ **Keyboard Support** - Press Enter to add or save tasks
- 🔒 **XSS Protection** - Safe HTML escaping for proper security

## How to Use

1. **Open the App** - Open `index.html` in your web browser
2. **Add a Task** - Type in the input field and click "Add Task" or press Enter
3. **Complete a Task** - Check the checkbox next to a task to mark it as complete
4. **Edit a Task** - Click the "Edit" button, modify the text, and click "Save"
5. **Delete a Task** - Click the "Delete" button and confirm the deletion
6. **Filter Tasks** - Use the filter buttons to view specific task categories

## Files

- **index.html** - Main HTML structure
- **style.css** - Styling and responsive design
- **script.js** - JavaScript functionality and logic

## Technical Details

- **Storage** - Tasks are stored in browser's LocalStorage for persistence
- **No Dependencies** - Pure vanilla JavaScript, no frameworks or libraries required
- **Browser Support** - Works on all modern browsers

## Keyboard Shortcuts

- **Enter** - Add a new task or save edits
- **Ctrl/Cmd + A** - Select all text in input fields

Enjoy organizing your tasks! 🚀
